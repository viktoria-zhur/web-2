Информация о API:

  • GET /lab7/rest-api/films/ - получить все фильмы
  • GET /lab7/rest-api/films/<id> - получить фильм по ID
  • DELETE /lab7/rest-api/films/<id> - удалить фильм по ID
  • PUT /lab7/rest-api/films/<id> - обновить фильм
  • POST /lab7/rest-api/films/ - добавить новый фильм
